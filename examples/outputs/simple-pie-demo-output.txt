📋 VizSeed DSL:
{
  "data": {
    "rows": [
      {
        "category": "移动端",
        "value": 45
      },
      {
        "category": "桌面端",
        "value": 35
      },
      {
        "category": "平板端",
        "value": 20
      }
    ],
    "fields": [
      {
        "name": "category",
        "type": "string",
        "role": "dimension",
        "nullable": false,
        "unique": true,
        "isDiscrete": true
      },
      {
        "name": "value",
        "type": "number",
        "role": "measure",
        "nullable": false,
        "unique": true,
        "aggregation": "sum",
        "isDiscrete": false
      }
    ]
  },
  "chartConfig": {
    "mapping": {
      "category": "category",
      "value": "value"
    },
    "type": "pie"
  },
  "transformations": [],
  "visualStyle": {
    "title": "用户访问设备分布"
  }
}

🎨 VChart Spec (可直接渲染):
{
  "type": "pie",
  "categoryField": "category",
  "valueField": "value",
  "data": [
    {
      "category": "移动端",
      "value": 45
    },
    {
      "category": "桌面端",

    "visible": true,
    "orient": "right",
    "position": "middle"
  },

    "visible": true
  },
  "tooltip": {
    "visible": true

✨ 核心流程总结:
1️⃣ 准备数据数组
2️⃣ new VizSeedBuilder(data)
3️⃣ 链式配置图表类型和字段
4️⃣ buildSpec() 生成规范
🎊 完成！可直接用于VChart渲染
